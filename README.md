![Game of Life](./public/Game_of_Life.png){width="500" height="500" style="display: block; margin: 0 auto"}

## Pre-requisites
* ```cmake```
* C compiler of your choice (```gcc```, ```clang```)

## Build & Run
```bash
mkdir build
cd ./build
cmake ..
cmake --build .

./Game_of_Life
```
